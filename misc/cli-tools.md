## CLI Tools

### Local Tools

- `git`: version control
- `vim`: text editor
- `ranger`: directory explorer
- `brew`: mac package manager

### Core Tools

- `df -h`: displays storage usage
- `w` & `who`: shows active ssh connections
- `tail -f`: Prints logs as they come in
- `man`: manual pages
- `less`: pager
- `grep`: search 
- `wc`: line counter
- `pwd`: print current directory
- `ssh`
- `cat`
